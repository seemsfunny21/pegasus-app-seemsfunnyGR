PEGASUS 224 - Permanent Local Storage Boot

- Replaced the previous background/video cache warmup logic with a single permanent local storage pipeline.
- On page open, PEGASUS downloads all known system/static files plus all known workout media paths into Cache Storage.
- The initialization loader now shows permanent local storage progress as a percentage and file count.
- After completion, same-origin files run cache-first from local storage.
- Old pegasus-shield and pegasus-videos caches are purged by the new service worker.
- Removed per-video background cache queues that caused READY/WAIT storms.
- Missing/slow videos do not fall back to warmup; PEGASUS waits for the requested local video.
- Package remains code-only: no images/ folder and no videos/ folder.
