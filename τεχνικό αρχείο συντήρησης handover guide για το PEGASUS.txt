21. Σημεία που είχαν βγάλει bug κατά το refactor

Αυτά είναι σημαντικά για να μη γυρίσουμε πίσω στα ίδια λάθη.

Bug 1: DOM exercise refs overwrite

Πρόβλημα:
το engine ή bridge έγραφε πάνω στο DOM exercises array σαν να ήταν serializable data.

Αποτέλεσμα:

οι ασκήσεις φαίνονταν
και μετά εξαφανίζονταν
και έδειχνε recovery day

Μάθημα:
το DOM exercise array δεν πρέπει να treated σαν κανονικό persisted state.

Bug 2: timer patch έσβηνε workout arrays

Πρόβλημα:
timer-only patch έκανε overwrite στο workout runtime.

Αποτέλεσμα:
instant finish ή σπασμένο Start.

Μάθημα:
timer patches πρέπει να αγγίζουν μόνο timer/progress fields.

Bug 3: test false fail λόγω μικρού action window

Πρόβλημα:
το test κοίταζε πολύ μικρό recent action history.

Αποτέλεσμα:
λειτουργικά pass αλλά test fail.

Μάθημα:
σε replay/history tests κοίτα μεγαλύτερο action window ή semantic sequence μετά από αυτοματοποιημένα clicks.

24. Τελική σύνοψη

Το PEGASUS πλέον είναι:

πιο σταθερό
πιο δομημένο
πιο εύκολο στο debugging
